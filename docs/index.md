# DREAM TEAM Project

> Learn **Python**, **GitHub**, **Git**, and **Cooperation**!

## About

It was created as part of Script Languages course on [Wrocław Univeristy of Science and Technology](https://pwr.edu.pl/en/). The goal of it is to learn how to create projects with **Python**, use **GitHub** and **Git** version control system, and learn cooperation in a team.

## Features

- **Team tools** for data manipulation and display
- **Utility tools** for greeting people
